## نکات مهم

1. **فرمت تصاویر**: از فرمت JPG یا PNG استفاده کنید. JPG برای عکس‌ها و PNG برای تصاویر با پس‌زمینه شفاف مناسب است.

2. **ابعاد تصاویر**: 
   - تصاویر دوره‌ها و خدمات: 400x300 پیکسل (نسبت 4:3)
   - تصاویر اساتید: 300x300 پیکسل (مربع)
   - تصاویر پروژه‌های دانشجویان: 400x300 پیکسل
   - تصاویر وبلاگ: 800x400 پیکسل (نسبت 2:1)

3. **بهینه‌سازی**: قبل از آپلود، تصاویر را بهینه کنید تا حجم فایل کم شود و سایت سریع‌تر بارگذاری شود. می‌توانید از ابزارهایی مثل [TinyPNG](https://tinypng.com) استفاده کنید.

4. **نام فایل‌ها**: نام فایل‌ها باید دقیقاً همان باشد که در فایل‌های داده مشخص شده است.

5. **پس از افزودن تصاویر**: بعد از قرار دادن تصاویر، سایت را refresh کنید تا تغییرات نمایش داده شوند.

## منابع رایگان تصاویر

اگر نیاز به تصاویر دارید، می‌توانید از سایت‌های زیر استفاده کنید:
- [Unsplash](https://unsplash.com) - تصاویر با کیفیت بالا و رایگان
- [Pexels](https://www.pexels.com) - تصاویر و ویدیوهای رایگان
- [Pixabay](https://pixabay.com) - تصاویر، ویدیو و موزیک رایگان

حتماً مطمئن شوید که مجوز استفاده از تصاویر را دارید!
